
# ESP32 Smart Weather Station

An IoT weather monitoring system built using ESP32 and DHT sensors that displays real-time temperature and humidity on a web dashboard.

## Features
- Real-time temperature monitoring
- Humidity monitoring
- ESP32 web server
- Simple dashboard

## Hardware
- ESP32
- DHT11 / DHT22 sensor
- Breadboard
- Jumper wires

## Connection
DHT Data -> GPIO 4

## Setup
1. Install Arduino IDE
2. Install ESP32 board package
3. Install DHT library
4. Upload code
5. Open Serial Monitor
6. Visit ESP32 IP address
